# DNSTT Messenger (فارسی)

DNSTT Messenger یک پیام‌رسان با رمزنگاری سرتاسری (E2E) است که می‌تواند در شبکه‌های محدود از تونل DNS (با MasterDnsVPN) استفاده کند.

## شروع سریع (کاربر)

1. کلاینت MasterDnsVPN را دانلود و اجرا کنید.
2. فایل `client_config.toml` را تنظیم کنید (دامنه و کلید رمزنگاری را از مدیر سرور بگیرید).
3. MasterDnsVPN را روشن نگه دارید.
4. کلاینت DNSTT Messenger را اجرا کنید.
5. داخل برنامه وارد **Settings** شوید و سرور/پروکسی/زبان را تنظیم و ذخیره کنید.
6. ثبت‌نام یا ورود انجام دهید.

## راه‌اندازی سرور شخصی

1. رکوردهای DNS دامنه تونل را تنظیم کنید.
2. سرور MasterDnsVPN را اجرا کنید.
3. سرور پیام‌رسان (`server`) را با `config.json` اجرا کنید.
4. این اطلاعات را به کاربران بدهید:
   - `server_addr`
   - دامنه تونل
   - کلید رمزنگاری تونل

## ساخت (Build)

- کلاینت/سرور Go: `go build`
- کلاینت Electron: `npm install && npm start` (یا `npm run build:*`)
- کلاینت Android: `./gradlew assembleRelease` (یا Android Studio)

## نکات امنیتی

- کلید رمزنگاری MasterDnsVPN را محرمانه نگه دارید.
- از رمز عبور قوی استفاده کنید.
- زیرساخت سرور قابل‌اعتماد انتخاب کنید.
- در صورت احتمال نشت، کلیدها را فوراً تغییر دهید.
