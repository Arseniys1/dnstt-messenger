# DNSTT Messenger (Русский)

DNSTT Messenger — это мессенджер со сквозным шифрованием (E2E), который может работать через DNS-туннель (MasterDnsVPN) в условиях ограниченного интернета.

## Быстрый старт (клиент)

1. Скачайте и запустите клиент MasterDnsVPN.
2. Настройте `client_config.toml` (домен и ключ шифрования от администратора сервера).
3. Запустите MasterDnsVPN и оставьте его включенным.
4. Запустите клиент DNSTT Messenger.
5. Откройте **Settings** (доступно внутри приложения), задайте сервер/прокси/язык и сохраните.
6. Зарегистрируйтесь или войдите.

## Свой сервер

1. Настройте DNS-записи для туннельного домена.
2. Запустите сервер MasterDnsVPN.
3. Запустите сервер мессенджера (`server`) с `config.json`.
4. Передайте пользователям параметры подключения:
   - `server_addr`
   - домен туннеля
   - ключ шифрования туннеля

## Сборка

- Go клиент/сервер: `go build`
- Electron клиент: `npm install && npm start` (или `npm run build:*`)
- Android клиент: `./gradlew assembleRelease` (или Android Studio)

## Безопасность

- Храните ключ шифрования MasterDnsVPN в секрете.
- Используйте сложные пароли аккаунтов.
- Используйте доверенную серверную инфраструктуру.
- При подозрении на компрометацию меняйте ключи.
